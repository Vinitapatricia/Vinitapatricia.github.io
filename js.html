Overview

Typing with Hann is a web game that we purpose to join a competition. This web game is required users to be able to type a given sentence within the available time. The sentences that we use are based on everyday English. The sentences used are a collection of motivations and advice on current issues.

With this web game, we are hoping many people, especially students and edlery can improve their skill of typing. During make this web game, we were learn a lot about how backend works and research about the current problem around us.
